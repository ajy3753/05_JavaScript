jQuery

기존에 자바스크립트 언어만으로 구현하기에는 복잡했던 구문을 간소화하기 위해 만들어진 JavaScript 기반의 라이브러리
즉, JavaScript로 유용한 함수나 내용들을 다른 누군가가 이미 정의해 놓은 것.
복잡한 JavaScript 구문보다 짧고 쉽게 개발이 가능하다

* 장점

1) DOM 요소와 관련된 스크립트를 보다 쉽게 수현할 수 있음
2) AJAX(비동기 통신) 이벤트 처리 등을 폭넓게 지원함
3) jQuery와 관련된 확장형 플러그인, 오픈 API 등을 지원 (차트, 슬라이드, 캘린더...)

jQuery 라이브러리 연결 방법

head 태그 내에 script 태그로 외부 jQuery 관련 .js 파일 연결하기
단, 오프라인 / 온라인 방식이 있음
1. jQuery 라이브러리 직접 다운로드 받은 후 경로지정(오프라인)
2. CDN 방식(온라인)
: 파일을 다운로드 받아서 저장할 필요 없이, 라이브러리를 제공하는 사이트 url을 제시하여 사용

jquery 구문 작성하기

jQuery 구문으로 DOM 요소를 다루는 구문을 기본적으로 작성할 것이기 때문에 문서상의 요소들이 다 만들어지고 나서 실행되어야한다. 

      > 순수 자바스크립트에서는
      // document.onload 는 삭제됨
      window.onload = function() {
         실행내용;
      }

      > jQuery document.load -> window.onload
      해당 문서의 요소들이 전부 로드 되고 외부 문서들도
      다 가지고 온 후에 실행됨
